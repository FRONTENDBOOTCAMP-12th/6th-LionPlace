# 6조(사자6성) - LionPlace

멋쟁이사자처럼 프론트엔드 부트캠프 12기 1차 프로젝트

## 팀 멤버

| 이름   | 역할          | 맡은 페이지      |
| :----- | :------------ | :--------------- |
| 오지영 | 조장          | 지도, 리뷰       |
| 이형주 | 스크럼 마스터 | 회원관리         |
| 신은비 | 조원          | 저장, 피드       |
| 이우빈 | 조원          | 예약, 주문, 쿠폰 |

## 사용 기술 및 도구

- HTML5
- Pure CSS
- Javascript(ES6)
- Lit
- 빌드 도구: Vite
- 배포 도구: Vercel
