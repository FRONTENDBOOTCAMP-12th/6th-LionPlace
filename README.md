# 6조(사자6성🦁) - LionPlace

멋쟁이사자처럼 프론트엔드 부트캠프 12기 바닐라 프로젝트<br><br>

## 01. 프로젝트 소개
원하는 장소를 저장하고 방문했던 장소에 리뷰를 작성할 수 있는 모바일 웹브라우저입니다.<br>

방문했던 가게들을 쉽게 관리할 수 있어 편리한 리뷰 작성과 예약이 가능하며<br>

팔로잉한 사람들의 리뷰를 볼 수 있고, 소통하며 장소 정보를 공유할 수 있습니다!<br><br>
